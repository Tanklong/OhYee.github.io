解决自建标签渲染问题

Hexo 有一个非常方便的功能 - 自建标签

其使用如下
{% raw %} 123 {% endraw %}
这样 123 会以 HTML 的形式插入到网页里,而非 Markdown

许多主题也都加入了许多有用的 tag
比如 {% cq %} blah blah blah {% endcq %}
